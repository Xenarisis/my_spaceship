# Welcome to My Spaceship
***

## Task
The challenge is to return informations relates to our spaceship in function of the letters we give in a string to the function.

## Description
In the program it will take a string with letters in it who are 'R', 'L', 'A'.
After the string given are taken it will execute actions for each letters in the string that will modify the informations of the spaceship, 
then once each letters has been succesfuly passed the informations will be return.

## Installation
There is no installation to do for this project

## Usage
'R' is to turn rigth
'L' is to turn left
'A' is to advance

Use the function in the file, for example:
```
my_spaceship("RRAALRRLLLAAAA")
```

And if you want to have the result in the terminal use the function console.log for example: 
```
console.log(my_spaceship("RRAALRRLLLAAAA"))
```
